# Recursion

- One branch recursion example: Factorial
  - n! = n*(n-1)*(n-2)*...*1
  - n! = n*(n-1)!
  - O(n) time complexity
  - O(n) space complexity: call stack
  - Base case in recursion is the smallest subproblem
```python
def factorial(n):
  if n == 0:
    return 1
  return n*factorial(n-1)
```
- Recursion is about subproblems
